# Telegram гид-бот

## Sprotect Bots / Guide Platform

**guide.sprotectbots.ru**

Разработал Telegram-бота и CMS-платформу для создания фото- и аудиогидов с онлайн-оплатой внутри Telegram.

Платформа позволяет создавать цифровые экскурсии с маршрутами, локациями, фотографиями и аудиофайлами. Пользователь оплачивает доступ к закрытому контенту прямо в Telegram, а контент-менеджеры управляют материалами, транзакциями и поступлениями через веб-интерфейс.

## Архитектура

### Telegram Bot Layer

Бот на PyTelegramBotAPI принимает входящие сообщения и выступает gateway-слоем для пользовательского сценария.

### Application Layer

Django-приложение:

* обрабатывает бизнес-логику
* хранит данные пользователей
* работает с медиафайлами
* формирует ответы пользователю
* управляет оплатами и доступом к контенту

### Data Layer

PostgreSQL хранит пользователей, заказы, контент и состояние сценариев.

### Reverse Proxy Layer

NGINX отвечает за маршрутизацию и работу с внешним трафиком.
