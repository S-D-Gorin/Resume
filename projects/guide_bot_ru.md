## Telegram гид-бот

### Sprotect Bots / Guide Platform

**guide.sprotectbots.ru**

Разработал Telegram-бота с поддержкой онлайн-оплаты и полноценной CMS-платформой для создания фото и аудиогидов.

Платформа позволяет создавать цифровые экскурсии, с отдельными локациями, фото и аудиофайлами. Оплачивать доступ к закрытой части контента можно внутри Telegram. А управлять контентом, отслеживать транзакции и поступления, можно через веб-интерфейс.

## Архитектурная схема

### 1. Telegram Bot Layer

Бот на PyTelegramBotAPI принимает входящие сообщения и выступает как gateway / proxy слой.

### 2. Application Layer

Django-приложение:

* обрабатывает бизнес-логику
* хранит данные пользователей
* работает с медиа
* формирует ответы пользователю
* управляет оплатами

### 3. Data Layer

PostgreSQL для хранения пользователей, заказов, контента и состояния сценариев.

### 4. Reverse Proxy Layer

NGINX для маршрутизации, SSL и работы с внешним трафиком.
