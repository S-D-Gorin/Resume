# Bot Sprotect

## Sprotect Bots
**sprotectbots.ru**

Антиспам-платформа для Telegram-групп.

Разработал Telegram-бота для автоматической модерации, удаления нежелательного контента и защиты групп от спама, рекламы, флуда и подозрительной активности.

Проект построен как контейнерная микросервисная платформа с разделением Telegram transport-слоя, backend-логики, данных, reverse proxy и фоновых задач.

## Архитектура

### Bot Gateway Layer

Отдельный контейнер с PyTelegramBotAPI:

* принимает события Telegram через `infinity_polling()`
* сериализует входящие сообщения
* отправляет события в backend по API в JSON-формате

Такой подход изолирует Telegram transport-слой от бизнес-логики.

### Backend Application Layer

Django-приложение под Gunicorn:

* обрабатывает входящие сообщения
* применяет антиспам-логику
* модерирует чаты
* работает с данными админ-панели
* отправляет ответы пользователям
* удаляет сообщения через Telegram API

Backend использует тот же `BOT_TOKEN`, что позволяет выполнять управляющие действия напрямую: `deleteMessage`, `sendMessage` и moderation actions.

Дополнительно используются внешние Python / FastAPI сервисы для расширенной обработки сообщений.

### Data Layer

PostgreSQL хранит:

* данные групп
* настройки модерации
* пользователей
* события
* служебные данные

### Reverse Proxy Layer

NGINX отвечает за входной трафик, reverse proxy и маршрутизацию запросов.

### Scheduler Layer

Cron-контейнер выполняет периодические задачи, рассылки, очистку данных и вызовы backend endpoints.
