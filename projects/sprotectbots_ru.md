Sprotect
Описание:
антиспам-бот для Telegram-групп, предназначенный для автоматической модерации и защиты чатов от нежелательного контента.

Проект построен на контейнерной архитектуре с разделением бота и backend-логики. Telegram-бот принимает и проксирует события, серверное приложение обрабатывает сообщения, применяет антиспам-правила и управляет логикой модерации.

стек:
docker
python
django + gunicorn
django-rest-api
pytelegrambotapi
nginx
postgres
docker
cron

Контейнеры:
1. Бот PyTelegramBotAPI который выступает в роли прокси. Принимает запросы infiniti_pooling() и шлет json.dumps({"message": message.json}) в Django 
2. Приложение: django принимает json и обрабатывает его. 
Тут выполняется вся логика, получаем, обрабатываем сообщения, управляем данными из админки. Также, в этом контейнере есть pytelegrambotapi с тем  же BOT_TOKEN что и в контейнере сботом, что по этому управление сообщениями (удаление и отправка) происходит в этом же контейнере. Для обработки сообщений используются внешние сервисы (python, fastapi) вызываемые Django.
3. БД Postgres
4. NGINX который стоит на входе в этот docker compose
5. CRON переодически обащается к django с запросами на рассылки и удаление сообщений
